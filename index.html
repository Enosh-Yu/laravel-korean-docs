라라벨 한국어 메뉴얼

이 문서는 라라벨의 공식 메뉴얼을 한국어로 번역하여 제공합니다. 라라벨을 이용하는데 언어로 인해서 겪는 문제점들을 해소하기 위해서 마련되었습니다.

라라벨?

라라벨은 PHP 에서 사용할 수 있는 가장 모던하고 세련된 프레임워크입니다. php로 개발을 진행하는데 필요한 많은 기능들을 제공하고 있습니다. 라라벨을 사용하기 위해서는 다음과 같은 필요사항들이 있습니다

  • PHP 버전 5.4이상
  • Mcrypt PHP Extension 필요
  • OpenSSL PHP Extension 필요
  • Mbstring PHP Extension 필요
  • Tokenizer PHP Extension 필요

한국어 문서의 번역

라라벨의 한글 매뉴얼은 xpressengine 개발팀의 주도하에 많은 분들의 공헌으로 진행되었습니다. 오타나 수정사항에 대해서는 @findstar 에게 연락하시거나 github 에서 Pull Request 를 보내주시면 검토후 반영됩니다.

지원

라라벨 한국어 매뉴얼은 Naver D2 와 Xpressengine 개발팀의 지원하에 진행되었습니다.
